Code Interpreter

This Code Interpreter understands the following instructions:

mov x y - copies y into register x

inc x - increases the content of the register x by one

dec x - decreases the content of the register x by one

jnz x y - jumps to an instruction y steps away (positive forward, negative backward), but only if x is not zero

add x y - add the content of the register x with y and stores the result in x

sub x y - subtract y from the register x and stores the result in x

mul x y - multiply x by y and stores the result in x

div x y - divide x by y and stores the result in x

label: - define a label position eg. start:

cmp x y - compares x and y. Result used in conditional jumps (jne, je, jge, jg, jle and jl)

jne label - jump to the label if the values of the previous cmp command were not equal.

je label - jump to the label if the values of the previous cmp command were equal.

jge label - jump to the label if x was greater or equal than y in the previous cmp command.

jg label - jump to the label if x was greater than y in the previous cmp command.

jle label - jump to the label if x was less or equal than y in the previous cmp command.

jl label - jump to the label if x was less than y in the previous cmp command.

The outputted results will appear here when you click run
